# StashJar

Habit-driven savings app with a mint-glass fintech UI, daily challenge loops, and Base-aligned funding flows.

![StashJar home screen](./apps/pwa/public/stashjar-home-preview.png)

## Apps

- `apps/pwa` - Next.js PWA frontend
- `services/ledger-service` - ledger and account operations
- `services/onchain-service` - onchain and reconciliation workers
- `contracts/stash-vault` - smart contract workspace
